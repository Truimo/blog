# Blog
Just a blog, under construction, use [notion](https://www.notion.com/) as backend server.

## Getting Started
This is a [React Router](https://reactrouter.com/home) project.

Run in production server:

- build application:
```bash
pnpm run build
```
- start server:
```bash
pnpm run start
```

Run in development server:
```bash
pnpm run dev
```
